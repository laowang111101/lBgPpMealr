# 项目简介 java1113

本项目是基于若依框架开发的前后端分离的统一权限管理平台，具备完善的功能模块，致力于提供一站式的系统管理解决方案。

## 技术栈

- 前端：Vue
- 后端：Spring Boot
- 数据库：MySQL
- 缓存：Redis
- 消息队列：RabbitMQ

## 功能模块

- **用户管理**：包括用户信息的增删改查等操作。
- **角色管理**：角色权限的分配与维护。
- **菜单管理**：系统菜单的配置与权限设置。
- **部门管理**：部门结构的管理与维护。
- **岗位管理**：岗位信息的设置与调整。
- **字典与参数管理**：系统字典及参数的维护。
- **系统公告**：公告信息的管理与发布。
- **日志追踪**：系统操作的日志记录与追踪。

### 系统监控

- **在线用户管理**：查看和管理当前在线用户。
- **定时任务管理**：系统定时任务的管理与调度。
- **系统状态监控**：实时监控系统状态。
- **缓存系统监控**：监控Redis缓存使用情况。

## 系统角色

- 系统管理员：负责系统整体配置与维护。
- 普通用户：进行日常业务操作。

## 运行环境

- 建议配置：64位操作系统，4GB以上内存，2GHz以上CPU。

## 部署步骤

1. 准备所需环境：Java环境、MySQL数据库、Redis服务、RabbitMQ服务。
2. 导入数据库脚本。
3. 部署后端服务。
4. 部署前端服务。
5. 启动系统。
